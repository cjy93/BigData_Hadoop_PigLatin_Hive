First create a table in the database nsdba2_superstore and then create the table “weather”.
```

CREATE TABLE nsdba2_superstore.weather(
temp_date TIMESTAMP,
temp_year INT,
temp_month INT,
temp_max DOUBLE,
temp_min DOUBLE
)
ROW FORMAT DELIMITED
FIELDS TERMINATED BY ','
LOCATION '/bakeinc/weather';
```
